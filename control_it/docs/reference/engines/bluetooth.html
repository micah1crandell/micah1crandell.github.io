Bluetooth Web Bridge

The Bluetooth Web Bridge is a specialized engine that allows the iPhone to broadcast actions to a connected web browser or other BLE central device.

Use Cases

Relaying Actions: Use a laptop or desktop computer to relay HTTP requests from your phone. This is useful when the phone is on a cellular network but needs to control devices on the computer's local network (LAN).

How it works

  1. The website (or central) connects to the iPhone (Peripheral) via Bluetooth Low Energy.
  2. When you fire an HTTP action on the iPhone, it broadcasts the details (Target IP, Path, Body) via a Notification.
  3. The connected website receives the data and can choose to execute the request locally.

Protocol Specifications

Service UUID

C0EECD00-1001-1001-1001-C0EECD000001

Characteristic

C0EECD00-1001-1001-1001-C0EECD000002 (Notify/Read)

Payload Limit

BLE MTU limits may require data fragmentation, though Control It attempts to fit payloads within standard limits. Large bodies may be truncated.
