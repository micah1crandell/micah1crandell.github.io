MIDI Engine

The MIDI Engine separates Control It from basic MIDI apps. It allows the app to process incoming MIDI commands and execute complex workflows entirely in the background, without any UI interaction.

Hardware Routing

The engine listens to all connected CoreMIDI sources by default.

USB MIDI

Plug & Play via the Camera Connection Kit. No driver configuration needed.

Bluetooth MIDI

Low latency wireless connection. Supports "Central" mode to connect to peripheral devices (like Yamaha wireless adapters).

Network MIDI

RTP-MIDI over WiFi. Requires a stable 5GHz network for low jitter.

Trigger Logic

Value Ranges

You can configure triggers to fire only when a value falls within a specific range. This is useful for:

  • Fader Zones: Split a single fader into multiple commands (e.g., Value 0-10 triggers "Low", 11-127 triggers "High").
  • Velocity Sensitive: Trigger different actions based on how hard you hit a drum pad.

Modes

Mode Description
On Press Fires when value > 0 (Note On). Standard behavior.
On Release Fires when value = 0 (Note Off). useful for "momentary" effects.
Continuous Passes the raw value (0-127) directly to a variable (e.g., controlling a Hue brightness slider).

Background Audio & Keep-Alive

Critical: iOS aggressively suspends background apps to save battery. To ensure the MIDI Engine runs reliably while your phone is locked:
  1. Go to Settings in Control It.
  2. Enable "Background Audio".
  3. The app will play a silent audio track. This flags the app as a "Media Playback" app to the OS, granting it indefinite background runtime.
