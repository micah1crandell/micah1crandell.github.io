The Shop (Community Marketplace)

The Shop is a centralized repository where users can publish, discover, and download community-created grids. It allows you to share your work or quickly bootstrap a new control surface.

Search & Discovery

The Shop uses CloudKit's query capabilities to help you find content.

Cloud Search

Search queries are sent to the server to find matching Titles. The search supports prefix matching (e.g. "Stream" matches "StreamDeck").

Categories

Grids are organized into manually selected categories like Home, Work, Audio, Video, Lighting, and MIDI.

Grid Validation

To ensure a healthy ecosystem, uploads are validated client-side:

  1. Size Check: Grids must be under 1MB in size. This limit ensures fast downloads for everyone. Large embedded images are the most common cause of hitting this limit.
  2. Format: The grid must be a valid exportable entity (JSON structure).
Security Note: The app does not modify your grid data during upload. Any API keys, passwords, or personal IPs included in your actions will be published. Always review your grid before uploading.

Versioning

The Shop infrastructure supports versioning, allowing creators to push updates to their grids without breaking existing downloads.
