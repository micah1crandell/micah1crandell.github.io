Publishing to The Shop

The Shop is a community marketplace where users can share their custom grids. Published grids are available for anyone to download and use.

Before you Publish

Check your IP Addresses: Remember that your actions likely contain local IP addresses (e.g., 192.168.1.50). Users who download your grid will need to update these. Consider adding a "README" action or clear labels.
Sanitize Secrets: Ensure you aren't accidentally sharing API keys or passwords in the headers or body of your actions. The app does not automatically strip these values.

Step-by-Step Guide

1. Prepare your Grid

Ensure your grid has a clear Name. You can edit this in the Grid Settings.

2. Upload to Shop

  1. Go to the Shop tab in the main app.
  2. Tap the Upload button (or "Publish Grid").
  3. Select the grid you wish to publish from the list.
  4. Fill in a Title, Description (Markdown supported), and select a Category.
  5. Tap Upload Grid.

3. Success

Once uploaded, your grid is immediately available in the Shop for other users to discover.

Updating a Grid

To update a grid you've already published, simply upload it again. The system detects if you are the creator and may create a new version or overwrite the existing one depending on the changes.

Removing a Grid

Currently, grids cannot be deleted manually from the app to prevent accidental data loss. If you need a grid removed, please contact support.
