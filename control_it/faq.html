FAQs

What is Control-It?

Control-It is an app designed to allow you to connect to and control various devices and services via protocols like OSC, HTTP, TCP, UDP, and MIDI. It's a customizable tool for managing connections and automating tasks.

How do I add a new connection?

You can add a new connection by navigating to the "Connections" section in the app and pressing the "Add Connection" button. You'll be able to specify the connection type, IP address, port, and any other necessary details.

Can I backup my data?

Yes! Control-It allows you to export all of your data as a backup file. This file can be imported later if needed, ensuring that your grids, connections, and actions are never lost.

Is Control-It available for Android?

Currently, Control It is only available for iOS devices. We are considering future support for Android, but there is no release date at this time.
