How to Use

Step 1: Setting Up

Download and install the Control It app from the App Store. The first step is to create a 'Connection'. Connections store the IP address and port of your target device. For example, if you want to control your Roku Smart TV, you would add the IP address of the TV and then set the port to 8060 to receive HTTP requests.

Step 2: Creating Actions

Click the '+' button beside or within the Actions view. This opens the Create Action page. Now you can begin creating your action. For example, if you want to create an action to mute and unmute your TV, you could title your Action Button 'Mute'. Then you select the Connection you created before. Now you can add the Action you are creating to a list. If you have not yet created a list, you can add it to the 'Default List'. Next, you can specify whether you want the Action to require confirmation before executing. Then hit save.

Step 3: Managing Actions

Organize your commands into lists and edit or delete them as needed. Use the export and import feature to share lists of commands across devices. The lists are exported as JSON files and can be edited within a text editor to easily add or remove Actions.

FAQ

How do I add a new action?

Click the '+' button beside the word 'Actions' on the home screen or within any list, fill in the required fields for your desired protocol, and save the action.

Can I export and import actions?

Yes, use the export feature to save your commands to a JSON file, and the import feature to load commands from a file. From here you can share lists with others.
